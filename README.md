# Portfolio - Senior Full Stack Developer

A premium, futuristic, and highly interactive portfolio website for a Senior Full Stack Developer. Built with a focus on performance, aesthetics, and smooth user experience.

## 🚀 Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Animations**: Framer Motion (motion/react)
- **State Management**: Zustand
- **Routing**: React Router 7
- **Icons**: Lucide React
- **Styling**: Glassmorphism, CSS Variables, Tailwind Utility Classes
- **Testing**: Jest, React Testing Library

## ✨ Key Features

- **Futuristic Dark Theme**: A clean, premium UI with glowing blue/purple gradients and glassmorphism effects.
- **Interactive Hero Section**: Animated background with floating shapes and a typing animation for roles.
- **About Me**: Detailed professional summary with key statistics (Years of Experience, Projects Delivered, Client Satisfaction).
- **Technical Expertise**: Categorized skills (Frontend, Backend, Mobile & Architecture, Tools & DevOps) with interactive hover effects.
- **Experience Timeline**: A vertical timeline showcasing professional growth and key achievements.
- **Project Showcase**: Case-study style presentation of major projects with detailed descriptions and tech stacks.
- **Services**: Specialized services offered, including Full Stack Development and System Design.
- **Contact Section**: Integrated contact form with validation and social links.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.

## 🛠️ Project Structure

- `src/components/`: Reusable UI components (Hero, About, TechStack, Experience, Projects, etc.)
- `src/constants/`: Static data and configuration (Project details, experience entries).
- `src/hooks/`: Custom React hooks.
- `src/store/`: Zustand state management.
- `src/index.css`: Global styles and Tailwind configuration.

## 🚦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Build for production:
```bash
npm run build
```
